## 🚀 Overview

Welcome to my personal portfolio repository! This project serves as a central hub for my professional identity, showcasing my skills, work experience, certifications, and featured projects. It is designed with a unique **Neobrutalist aesthetic**, combining stark contrasts, bold typography, and engaging 3D micro-interactions to create a memorable user experience.

## ✨ Key Features

- **🎨 Neobrutalist Design System**: Bold shadows, stark borders, and high-contrast dark mode for a visually striking aesthetic.
- **🪄 3D Hover Animations**: Interactive and dynamic UI elements using Framer Motion that respond beautifully to user interactions.
- **📱 Fully Responsive**: Pixel-perfect layouts ensuring a seamless experience across mobile, tablet, and desktop viewports.
- **⚡ Blazing Fast**: Built on Next.js App Router for optimal performance, SEO, and fast page loads.
- **♿ Accessible Components**: Utilizing Radix UI primitives to ensure high accessibility standards for all users.

## 🛠️ Tech Stack

This project is built using modern, cutting-edge web technologies:

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion (Framer Motion)](https://motion.dev/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Validation**: [Zod](https://zod.dev/) & React Hook Form
- **Analytics**: [Vercel Analytics](https://vercel.com/analytics)

## 💻 Getting Started

Want to run this project locally? Follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18+) and [pnpm](https://pnpm.io/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Naman1313/Naman-portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Naman-portfolio
   ```

3. **Install dependencies**
   ```bash
   pnpm install
   ```

4. **Start the development server**
   ```bash
   pnpm run dev
   ```

5. **Open in browser**
   Visit `http://localhost:3000` to see the magic happen! ✨
